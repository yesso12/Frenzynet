# Screenshot Capture Guide (Chrome Web Store)

## Target format
- Recommended: `1280x800` PNG
- Keep browser zoom at `100%`
- Use light clutter-free tab set

## Fast capture flow (Windows)
1. Open Chrome with extension loaded.
2. Open a supported streaming page.
3. Open DevTools (`F12`) -> Toggle device toolbar (`Ctrl+Shift+M`) OFF.
4. Resize browser window near 1280x800 viewport.
5. Use Snipping Tool (`Win+Shift+S`) rectangular capture.
6. Save as PNG with names:
   - `01-rail-open.png`
   - `02-set-party.png`
   - `03-chat-reactions.png`
   - `04-mic-cam.png`
   - `05-popup.png`

## Fast capture flow (macOS)
1. Open Chrome with extension loaded.
2. Open target page and show extension UI state.
3. Press `Cmd+Shift+4`, select region.
4. Save/rename files to the same sequence above.

## Quality checks before upload
- Text is readable at normal zoom
- No private user data visible
- No unrelated logos that imply third-party ownership
- Screenshot content matches listed features exactly
